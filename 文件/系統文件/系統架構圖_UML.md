# 系統架構圖（UML 模組化設計）

> 依據教授建議：定義系統邊界、模組間關係、Level 1 → Level 2 展開
> 方框 = 模組（Class / 功能主體）；虛線框 = 系統邊界（System Boundary）；箭頭 = 依賴 / 資料流

│  │ WaypointManager      │     │ ObservationBuilder               │ │
│  │                      │     │                                  │ │
│  │ 巡邏點（依優先級）：   │     │ · LiDAR 90 點正規化 /3.5           │
│  │  1. 電源控制室(+80)   │     │ · 目標距離 /5.0                  │ │

│  │  3. 生產線A   (+50)   │     │ · 線速度 /0.25  角速度 /1.5      │ │
│  │  4. 生產線B   (+30)   │     │ · 4 幀疊加 -> obs[376]          │ │
│  │  5. 出入口    (+20)   │     └──────────────────┬───────────────┘ │

│                                                                     │

## 系統邊界說明

| 分類 | 元件 | 說明 |
|------|------|------|
| **外部** | Gazebo 模擬器 | /scan /imu /odom，非本專題開發 |
| **外部** | OpenAI API | GPT-4o-mini / GPT-4o，非本專題開發 |
| **外部** | SROS2 / X.509 PKI | DDS 加密設定，非開發 |
| **內部** | SmartPatrolNode | SAC 推論 + 巡邏點管理（patrol_node.py）|
| **內部** | DDSSecurityMonitor | 白名單偵測 + 警報 + 緊急停止 |
| **內部** | MissionManager / SystemStatus | 任務整合層 |
| **內部** | LLM Fuzzer（離線） | 驗證模組 B 防禦強度，發現 31 筆漏洞 |
